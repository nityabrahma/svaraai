# **App Name**: LeadPilot AI

## Core Features:

- Automated Scraping: Scrape company/contact data from various sources using Playwright.
- AI-Powered Enrichment: Enrich lead data with LLM to extract insights and identify key contacts. LLM will use third party tool connectors such as Clearbit and Hunter.io
- Lead Validation: Validate lead data using email/phone validators, domain checks, and duplicate detection.
- Lead Scoring: Automatically assign a quality score to leads based on validation and enrichment data.
- Subscription Management: Manage freemium subscriptions using Stripe.
- Webhook Integration: Deliver high-quality leads to clients via webhooks.
- User Dashboard: SPA for dashboards, lead review, reports, and billing.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) for trust and professionalism.
- Background color: Light gray (#F5F5F5), providing a clean, professional backdrop.
- Accent color: Soft green (#34A853), creating contrast and signaling data quality and accuracy.
- Headline font: 'Space Grotesk', sans-serif, to convey a contemporary, techy feel. Body font: 'Inter', sans-serif.
- Use simple, modern icons to represent data points and actions.
- Clean and intuitive layout with a focus on data visualization.
- Subtle animations for data loading and updates to enhance user experience.
